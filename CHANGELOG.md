# 📝 更新日志 (Changelog)


## [v3.6] - 2026-03-18

### 🌏 语言本地化与 UI 细节加固
- **重构 🌏 深度语言本地化**：设置面板全模块（AI 增强、备份迁移、账号安全、API 配置、通知服务）及其交互提示语、占位符、操作说明完成 100% 中文化。
- **优化 🎨 UI/UX 纯净体验**：
    - 移除了设置项标题的斜体（Italic）与特粗（font-black）样式，使排版视觉重心更稳、更符合中文阅读习惯。
    - 术语表达去技术化：将“Engine”改为“配置/设置”，“Vault”改为“数据管理”，使非技术用户更易理解。
- **增强 🛡️ 账号安全交互**：优化了双重验证（2FA）设置流程的步骤指引，增强了密钥复制与二维码扫描的视觉引导。
- **优化 ⚙️ 协议配置反馈**：重置 Telegram API 协议及清除通知配置时新增了更明确的风险警示文案（中文原生）。


## [v3.5] - 2026-03-17

### ✨ 核心重构与交互升级
- **重构 💎 UI/UX 全面模态化**：由 Linear 风格驱动的自定义确认弹窗，彻底替换原生浏览器 `confirm()`。所有删除、重置、清空等敏感操作均拥有平滑的缩放动画与半透明磨砂背景。
- **新增 🕒 服务器时间同步**：仪表盘右上角及通知设置中增加实时服务器时间显示，帮助用户精准校准每日签到报告的执行计划。
- **新增 🧹 账户运行日志清理**：支持一键清空特定账户的全部历史日志，通过自定义“Purge”模态框进行安全确认，保持数据库整洁。
- **增强 🎇 交互动效与反馈**：
    - 新装“立即终止 (Terminate)”任务流确认界面，强化操作安全感。
    - 优化了神经同步 (Neural Sync) 数据的导入反馈，增加进度感官暗示。
- **优化 🎨 极简暗黑视觉**：微调了侧边栏与模态框的边框发光（Accent Glow）效果，使 Pure Dark 风格在不同屏幕下表现更一致。

## [v3.4] - 2026-03-16

### 🚀 矩阵管理与迁移增强
- **新增 ⚡ 会话迁移终端 (Session Migration)**：支持一键导出全量账号会话 ZIP 凭证。换机部署无需扫码，实现物理级克隆复活。
- **新增 🚀 批量任务分发 (Bulk Distribution)**：支持在任务看板中勾选多个账号，并一键分发“任务包”（JSON 配置）。一分钟拉齐百号矩阵。
- **优化 📈 任务状态雷达**：重构了任务列表的加载逻辑，支持实时订阅后端状态推送。
- **修复 🛠️ 内存碎片回收**：优化了长时间运行下的 session 句柄释放策略。
